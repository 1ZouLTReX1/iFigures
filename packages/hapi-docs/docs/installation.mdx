Download Node from [nodejs.org](https://nodejs.org/en/) (LTS version)

</TabItem>
<TabItem value="mac">

Download Node from [nodejs.org](https://nodejs.org/en/) (LTS version)

</TabItem>
<TabItem value="linux">

### If using Debian, Ubuntu, or Linux Mint

Click [here](https://www.geeksforgeeks.org/installation-of-node-js-on-linux/) for more info.

Install Node.js:

```bash
  sudo apt install nodejs
```

Install NPM:

```bash
  sudo apt install npm
```

### If using other linux distributions

Follow the instructions [here](https://nodejs.org/en/download/package-manager/) for your linux distro.

To verify that Node is installed:
```bash
  node -v
```

To verify that npm is installed:
```bash
  npm -v
```
